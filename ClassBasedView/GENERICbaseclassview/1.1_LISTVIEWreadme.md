# üß© Custom Query ‚Äî MOST IMPORTANT SKILL

## ‚≠ê get_queryset()

Override when logic depends on request.

---

### Example ‚Äî Filter by author

```python
def get_queryset(self):
    return Book.objects.filter(author="John")
```

---

### Example ‚Äî Use URL parameter

URL:

```
/books/john/
```

```python
path("books/<str:author>/", BookListView.as_view())
```

View:

```python
def get_queryset(self):
    author = self.kwargs.get("author")
    return Book.objects.filter(author=author)
```

---

### Example ‚Äî Search using GET

```
/books/?q=python
```

```python
def get_queryset(self):
    q = self.request.GET.get("q")
    if q:
        return Book.objects.filter(title__icontains=q)
    return Book.objects.all()
```

# ‚ö° Full Professional Example

```python
class BookListView(ListView):
    model = Book
    template_name = "books/book_list.html"
    context_object_name = "books"
    paginate_by = 5

    def get_queryset(self):
        q = self.request.GET.get("q")
        if q:
            return Book.objects.filter(title__icontains=q)
        return super().get_queryset()

    def get_context_data(self, **kwargs):
        context = super().get_context_data(**kwargs)
        context["title"] = "Book Library"
        return context
```
